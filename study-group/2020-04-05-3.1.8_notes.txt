
# upaḍḍhasuttaṃ #

"The Half Sutta"

Saṃyutta Nikāya, mahāvaggo n, 1. maggasaṃyuttaṃ, 1. avijjāvaggo n, 2. upaḍḍhasuttaṃ n (SN 45.2)

> Evaṃ me sutaṃ 
- thus it was heard by me

> Ekaṃ samayaṃ bhagavā sakyesu (loc. pl.) viharati
- one time the bhagavā was [among the sakyans] living

> nagarakaṃ (loc.) nāma sakyānaṃ nigamo (nom.)
- in a town named Nagaraka, a Sakyan market town

> Atha kho āyasmā ānando yena bhagavā tenupasaṅkami
- there Ananda then bhagavā [approached]

Āyasmant = venerable, old
yena / tena (relative/demonstrative pronouns)
tenupasaṅkami = tena (ta, he) + upasaṅkami (aor. approached)

> upasaṅkamitvā (abs.) bhagavantaṃ abhivādetvā (abs.) ekamantaṃ nisīdi (aor.).
- abs. "having": "having approached", "having greeted"
- after approaching the bhagavā and bowing he sat to one side (end)

ekamantaṃ = eka + anta (end)

> Ekamantaṃ nisinno kho āyasmā ānando (nom.) bhagavantaṃ (acc.) etadavoca:
- sitting on one side ananda spoke to the bhagava like this 

etadavoca = spoke like this 

> "upaḍḍhamidaṃ, bhante, brahmacariyaṃ, yadidaṃ– kalyāṇamittatā kalyāṇasahāyatā kalyāṇasampavaṅkatā"ti.
- this half of the holy life, bhante, which is this- virtuous [friends, companions, colleagues]

upaḍḍhamidaṃ = upaḍḍha (aḍḍha = half) + idaṃ (neu. this)
kalyāṇa - morally good, virtuous
mittatā - state of friendship (tā = abstract noun: "-ness", "-ship")
sahāyatā (sahāya) - companionship, friendship 
sampavaṅkatā - connection, friendship, intimacy

> "Mā hevaṃ, ānanda, mā hevaṃ, ānanda!
mā - don't / not
hevaṃ - (he + evaṃ) emphatic + that / so
- "certainly not so"

> Sakalamevidaṃ, ānanda, brahmacariyaṃ, yadidaṃ – kalyāṇamittatā kalyāṇasahāyatā kalyāṇasampavaṅkatā.
the whole life, ananda, of brahmacariya, this is - friendship, friendship, friendship.

sakala - whole
mevidam = eva (emphatic) + idaṃ (this)?

> Kalyāṇamittassetaṃ, ānanda, bhikkhuno (gen. / dat.) pāṭikaṅkhaṃ kalyāṇasahāyassa kalyāṇasampavaṅkassa– 
- with noble friends/companions/connections
- "-ssa" = gen. abs. in this case = "when"

Kalyāṇamittassetaṃ = Kalyāṇamitta + etaṃ (this)
pāṭikankha - expected
- "this is expected"

> ariyaṃ aṭṭhaṅgikaṃ maggaṃ bhāvessati, 
- bhāvessati = bhāvana (fut.)
- he will develop the noble 8-fold path

> ariyaṃ aṭṭhaṅgikaṃ maggaṃ bahulīkarissati.
- he will take it up seriously (lit. "make much")

bahulīkarissati = bahu + lī + karoti

> Kathañcānanda, bhikkhu kalyāṇamitto kalyāṇasahāyo kalyāṇasampavaṅko ariyaṃ aṭṭhaṅgikaṃ maggaṃ bhāveti,
- kathañcānanda = Kathaṃ (how) + cā (and) + ananda
- how ananda does a bhikkhu who has noble friends cultivate (bhāveti) the 8-fold noble path?

> ariyaṃ aṭṭhaṅgikaṃ maggaṃ bahulīkaroti?
- how does he take up the noble 8-fold path seriously?

> Idhānanda, bhikkhu sammādiṭṭhiṃ bhāveti vivekanissitaṃ virāganissitaṃ nirodhanissitaṃ vossaggapariṇāmiṃ. 
- here ananda a bhikkhu develops right view, dependent on: seclusion, dispassion, cessation - maturing in relinquishment
- bhāveti: "makes development" (causative)

vivekanissitaṃ = viveka (detachment, seclusion) + nissitaṃ (dependent on)
virāganissitaṃ = virāga (dispassion) + nissitaṃ (dependent on)
nirodhanissitaṃ = nirodha (cessation) + nissitaṃ (dependent on)
vossaggapariṇāmiṃ = vossagga (relinquishment) + pariṇāmiṃ (transformed into / matured to)

> sammāsaṅkappaṃ bhāveti vivekanissitaṃ ...
- right understanding / thought

> sammāvācaṃ bhāveti ...pe...     - right speech
> sammākammantaṃ bhāveti ...pe... - right action
> sammā-ājīvaṃ bhāveti ...pe...   - right livelihood
> sammāvāyāmaṃ bhāveti ...pe...   - right effort
> sammāsatiṃ bhāveti ...pe...     - right awareness
> sammāsamādhiṃ bhāveti           - right concentration

> ... vivekanissitaṃ virāganissitaṃ nirodhanissitaṃ vossaggapariṇāmiṃ.
- depenedent on detachment/seclusion, dispassion, cessation ... leading to relinquishment

> Evaṃ kho, ānanda, bhikkhu kalyāṇamitto kalyāṇasahāyo kalyāṇasampavaṅko ariyaṃ aṭṭhaṅgikaṃ maggaṃ bhāveti, 
- this is how ananda noble friends/acquaintances/colleagues help develop the noble 8-fold path
> ariyaṃ aṭṭhaṅgikaṃ maggaṃ bahulīkaroti.
- bahu+karoti = great development, make much

bhāveti = cultivates 
bahulīkaroti = produces, works seriously 

> Tadamināpetaṃ, ānanda, pariyāyena veditabbaṃ yathā sakalamevidaṃ brahmacariyaṃ, yadidaṃ:
> kalyāṇamittatā kalyāṇasahāyatā kalyāṇasampavaṅkatā.
- "-atā" (abstract noun)

Tadamināpetaṃ = tad/taṃ (that/thus) + iminā (in these ways) + api (and) + etaṃ (this)
pariyāyena - pariyaya = method, applied method, figurative language (yena = by this method)
veditabbaṃ - (vedita = experienced) should be known (fut. pass. part.)
yathā - "as" but here, "as how:"
sakalamevidaṃ = sakala (whole) + m + evidaṃ (eva + idaṃ) = very much this thing
brahmacariyaṃ - "holy life" (adj.)
yadidaṃ - "it is this:" / "this is how it is:"

> Mamañhi, ānanda, kalyāṇamittaṃ āgamma jātidhammā sattā jātiyā parimuccanti,

mamaṃ - (ins.) "by me"
mamañ = ahaṃ (I / me) + hi (because of)
kalyāṇamittaṃ - (accusative) "as a good friend"
āgamma - owing to / dependent on
jātidhammā = jāti + dhamma (subject to birth)
sattā = satta (a being, pl.)
jātiyā - (abl.) from birth 
parimuccati = pari + mutti - release, escape (present tense)

> jarādhammā sattā jarāya parimuccanti;
- beings subject to old age are liberated from old age

jarāya - (abl.) "from death"
jarā - old age

> maraṇadhammā sattā maraṇena parimuccanti;
- beings subject to death are liberated from death

maraṇena - (ins.) "with/by/through death"
maraṇa - death

NOTE: ablative and instrumental often have similar intentions / meanings ("with" vs. "from")

> sokaparidevadukkhadomanassupāyāsadhammāsattā sokaparidevadukkhadomanassupāyāsehi parimuccanti.

soka - grief
parideva - extreme wailing/crying 
dukkha - suffering 
domanassa - du + mana ("bad mind")
upāyāsa - disturbance, unsettling condition / strong grief (strongest of the list)
dhamma - condition
sattā - beings

-ehi - (pl. ins.) "from all these things"
parimuccanti - released from

> Iminā kho etaṃ, ānanda, pariyāyena veditabbaṃ yathā 
> sakalamevidaṃ brahmacariyaṃ, yadidaṃ– kalyāṇamittatā kalyāṇasahāyatā kalyāṇasampavaṅkatā”ti.

iminā - (ayaṃ) through this 
etaṃ - this (double pronoun)
pariyāyena - by this method (pariyāya)
veditabbaṃ - should be known / experienced 
yathā - that, then, according to (connector) - but here: "how"

sakalamevidaṃ etc. = the whole life of a brahmacari is noble friends/companions/connections 

Sudhakar asked about the definition of the "5 kinds of seclusion". Bhikkhu Bodhi lists them as:

(i) “in a particular respect” (tadaṅga, temporarily, by the practice of insight); 
(ii) by suppression (vikkhambhana, temporarily, by attainment of jhāna); 
(iii) by eradication (samuccheda, permanently, by the supramundane path); 
(iv) by subsiding (paṭippassaddhi, permanently, in fruition); and 
(v) by escape (nissaraṇa, permanently, in Nibbāna). 

(i) tad/taṃ (that/thus) + aṅga (limb/digit)
